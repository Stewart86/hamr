# Default Configuration Files

This directory contains reference configuration files for Hamr.

## config.json

The default configuration file with all available options set to their default values.

**Installation:**
```bash
# Copy to your config directory (modify as needed)
mkdir -p ~/.config/hamr
cp config/config.json ~/.config/hamr/config.json
```

Note: Hamr works with an empty config or no config file at all. All fields have sensible defaults. You only need to create a config file to customize specific settings.

For documentation on all config options, see [docs/getting-started/configuration.md](../docs/getting-started/configuration.md).
